    # 坐标系说明

    坐标系分为两种，地理坐标系与投影坐标系

    一般常见的是测量系统是WGS84（1984年全球测量系统）

    基于84坐标系的常用坐标为两种

    EPSG：4326

    EPSG：3857

    坐标系 投影 坐标系类型 单位 作用
    84坐标 4326 地理坐标系 度 GPS定位
    84坐标 3857 投影坐标系 米 平面地图
    CGCS2000 4490 地理坐标系

    84坐标系是美国定义的标准，2000坐标系是中国定义的，4326与4490都是经纬度坐标系，基本相同
